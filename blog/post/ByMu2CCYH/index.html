ChubaoFS Gave a Talk on Shanghai Qcon2019

October 17, 2019

As a maintainer of ChubaoFS, Shuoran Liu gave a talk “ ChubaoFS: An Open Source Cloud Native Distributed File System ” on Shanghai Qcon 2019. His talk focused on the next five topics:

  • How to implement multi-tenancy and improve storage resource utilization ratio; 
  • How to leverage dual replication protocols to support sequential write and overwrite;
  • How to support heavy workload of file system metadata operations; 
  • How to optimize for small files; 
  • How to trade off between POSIX semantics and performance.
